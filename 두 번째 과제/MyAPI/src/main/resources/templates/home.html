문제 1

두 수를 입력하면, 다음과 같은 결과가 나오는 GET API를 만들어 보자!

  • path : /api/v1/calc 이다.
  • 쿼리 파라미터 : num1, num2
{
"add": 덧셈결과,
"minus": 뺄셈결과,
"multiply": 곱셈결과
}

예시

GET/api/v1/calc?num1=10&num2=5

{
"add": 15,
"minus": 5,
"multiply": 50
}
